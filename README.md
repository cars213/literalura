# Literalura
Libreria desarrollada en java como desafio del curso Java y Spring Framework, obtenga libros de la API Gutendex y almacénelos en una base de datos Postgres SQL, a su vez puede realizar busquedas por nombre, listar los libros almacenados y filtrar autores por año vivo

<img width="522" height="383" alt="image" src="https://github.com/user-attachments/assets/a8317d72-36b5-43f9-8f56-1c6bc58214d5" />

## Descripción del Proyecto
Proyecto finalizado, es una libreria conectada a una API llamada gutendex, esta te permite buscar libros informacion de los autores y hasta la popularidad de los libros, en el menu puede acceder a diferentes opciones
iniciando por la busqueda de un libro en la api, puedes listar los libros que se han buscado en la base de datos, puedes listar todos los autores almacenados, puedes buscar un determinado año si los autores estan vivos en esa fecha
y filtrar los libros por idioma, por el momento solo estan disponibles español e ingles.


## :hammer:Funcionalidades del proyecto
- `Funcionalidad 1`: Seleccion menu de inicio.
- `Funcionalidad 2`: Comprobacion para evitar que el usuario ingrese letras u algo distinto a numeros en la seleccion del menu.
- `Funcionalidad 3`: Obtencion de los libros de la API gutendex.
- `Funcionalidad 4`: Almacenamiento de los libros buscado con base de datos Postgres SQL.
- `Funcionalidad 5`: Validacion que no se repitan libros al guardar en la base de datos
- `Funcionalidad 6`: Validacion que no se utilicen letras u otro caracter diferente a numero en la busqueda por año
- `Funcionalidad 7`: Validacion para evitar informacion invalida o vacia de la API a la base de datos.
